# NexBand

Monorepo layout:

- **`frontend/`** — Next.js app (UI)
- **`backend/`** — API server (add your stack here)

## Getting started

From the **NexBand root** folder (`NexBand/`):

```bash
npm install
npm run dev
```

This starts the frontend at [http://localhost:3000](http://localhost:3000).

### Run only the frontend

```bash
cd frontend
npm run dev
```

### Run only the backend

After you add a backend server:

```bash
cd backend
npm run dev
```

Or from the root:

```bash
npm run dev:backend
```
